Cardboard Box


Cardboard Box is a group project game that i worked on with my fellow classmates as part of our graded module assignment. This is a stealth game that utilizes a box to block light and to use this as a cover. The game involves camera system which must be avoided and also security lasers that can be turned off but interacting with the fuse box.

The game forces the player to hold the cardboard box throughout the whole game which helps in manoeuvring the environment through climbing and creating shadows to avoid being seen by the cameras.
